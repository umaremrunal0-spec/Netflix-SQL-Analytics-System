# Netflix User & Content Analytics System

## Project Overview
A SQL-based analytics project designed to analyze Netflix user behavior, content performance, and subscription trends.

## Tools Used
- MySQL
- MySQL Workbench

## Database Design
The project consists of five tables:

1. Users
2. Movies
3. Watch History
4. Ratings
5. Subscriptions

## Analytics Performed

- Most Active Users
- Highest Rated Movies
- Subscription Analysis
- Revenue Analysis
- Watch Time Analysis
- Genre Analysis

## SQL Concepts Used

- Primary Keys
- Foreign Keys
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY

## Project Outcome

The project demonstrates practical SQL skills used in real-world streaming platforms to analyze user engagement and content performance.
